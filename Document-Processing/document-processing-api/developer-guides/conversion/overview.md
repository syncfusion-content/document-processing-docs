# Conversion

The Syncfusion Document Processing Engine accepts multiple file formats as input, such as Word, Excel, PowerPoint, and HTML, and converts them into PDF documents.

## Key capabilities

* Word to PDF
* Excel to PDF
* Powerpoint to PDF
* HTML to PDF
