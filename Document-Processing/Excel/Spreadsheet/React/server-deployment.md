# Deploying Spreadsheet Server in Selected Deployment Environment

The Syncfusion React Spreadsheet relies on a server-side application to process Excel files. The server converts Excel files to the Spreadsheet JSON model for rendering in the client and converts the JSON model back to Excel files when saving. This  section explains how to deploy the spreadsheet server to user preferred environment and expose stable service URLs for the Spreadsheet to use.
