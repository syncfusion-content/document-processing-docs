# Organize Pages in ASP.NET Core PDF Viewer control

The PDF Viewer enables efficient document management by organizing pages seamlessly. The Organize Pages feature supports rotation, rearrangement, insertion, deletion, copying, and duplication of pages.

The Organize Pages feature provides the following options:

* **Rotate Pages:** Adjust page orientation in 90-degree increments.
* **Rearrange Pages:** Change page sequence with drag-and-drop.
* **Insert New:ages** â€” Add blank pages at desired positions.
* **Delete Pages:** Remove unnecessary pages.
* **Copy Pages:** Duplicate pages to reuse content.
* **Import PDF Document:** Merge pages from another PDF.
* **Select All Pages:** Apply bulk actions to every page.
* **Real-Time Updates:** Changes appear instantly in the viewer. Click **Save** to apply modifications.
* **Save As Functionality:** Click **Save As** to download a modified copy of the document.

To access the organize pages feature, simply open the PDF document in the PDF Viewer and navigate to the left vertical toolbar. Look for the `Organize Pages` option to begin utilizing these capabilities.
