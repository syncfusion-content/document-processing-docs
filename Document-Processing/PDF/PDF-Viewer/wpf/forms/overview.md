# Overview of Forms in WPF PDF Viewer

The Syncfusion WPF PDF Viewer delivers a complete,easy‑to‑use PDF forms experience.It allows form fields to be filled and edited directly within PDF documents. These actions are supported through both an intuitive user interface and powerful programmatic APIs, and form fields can be added and deleted through programmatic APIs alone.

The PDF Viewer also includes smooth import and export support for form data, making integration effortless. Developers benefit from extensive API control, while end users enjoy a clean and simple interface designed for a seamless and stress-free form-filling experience.

![WPF PDF Viewer Form Filling](./images/wpf-pdf-viewer-form-filling.png)

## Filling PDF Forms

In WPF PDF Viewer, effortless PDF form filling is supported through a clean and intuitive UI, as well as automated workflows powered by robust APIs. Flexible form data import and export capabilities ensure smooth and efficient operations when working with PDF forms.

See the [Filling PDF Forms](./form-filling) page for full details.
