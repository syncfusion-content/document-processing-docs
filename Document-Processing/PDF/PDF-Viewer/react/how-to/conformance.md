# Supported PDF conformance in React PDF Viewer

The React PDF Viewer supports the following PDF/A and PDF/X conformance levels:

* PDF/A-1a
* PDF/A-1b
* PDF/X-1a
* PDF/A-2a
* PDF/A-2b
* PDF/A-2u
* PDF/A-3a
* PDF/A-3b
* PDF/A-3u
* PDF/A-4
* PDF/A-4e
* PDF/A-4f
