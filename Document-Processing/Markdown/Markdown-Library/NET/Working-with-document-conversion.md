# Working with Conversion in Markdown Library

The Syncfusion Markdown library allows you to convert Markdown documents to various file formats and vice versa. This flexibility enables you to work with Markdown content and transform it according to your needs whether sharing documents, publishing to the web, or preserving content in different formats. The library supports bidirectional conversion between Markdown and other popular document formats.

The following conversions are supported by the Markdown library:

## Markdown to Word

Markdown to Word conversion allows you to convert an existing Markdown file to a Word document (DOCX, DOC, or RTF format). This conversion is useful when you need to edit or share Markdown content in a Word-compatible format. The Markdown library preserves text formatting, headings, lists, code blocks, tables, hyperlinks, and images during the conversion process. You can also customize the image handling and encoding settings during conversion.

For more details, refer to the [Markdown to Word conversion](https://help.syncfusion.com/document-processing/word/conversions/markdown-to-word-conversion) documentation.

## Word to Markdown

Word to Markdown conversion allows you to convert an existing Word document to a Markdown file. This conversion is useful when you need to share content in a lightweight markup format for version control systems, blogs, or documentation websites.

For more details, refer to the [Word to Markdown conversion](https://help.syncfusion.com/document-processing/word/conversions/word-to-markdown-conversion) documentation.
