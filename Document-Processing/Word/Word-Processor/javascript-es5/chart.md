## Supported Chart Types

The following chart types are supported in Document Editor
* Scatter_Markers
* Bubble
* Area
* Area_Stacked
* Area_Stacked_100
* Bar_Clustered
* Bar_Stacked
* Bar_Stacked_100
* Column_Clustered
* Column_Stacked
* Column_Stacked_100
* Pie
* Doughnut
* Line
* Line_Markers
* Line_Markers_Stacked
* Line_Markers_Stacked_100
* Line_Stacked
* Line_Stacked_100
